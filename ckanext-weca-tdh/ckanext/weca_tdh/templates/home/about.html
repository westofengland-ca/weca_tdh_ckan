    {{ _('What is the Transport Data Hub?') }}

    {% trans %}

    The Transport Data Hub (TDH) is a digital platform for searching, sharing, and analysing transport data. It ingests big datasets, supports automation and API integrations, and enables analysis and reporting through Power BI. TDH aims to optimise transport operations and services, helping users make better decisions, evaluate interventions, and plan initiatives. This supports the West of England Mayor's goal to improve public transport.
